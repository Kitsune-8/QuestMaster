Квест-Мастер

Описание

Квест-Мастер — это мобильное приложение для оперативного управления бронированиями и ресурсами квест-рума. Оно создано для замены неудобных бумажных журналов и разрозненных электронных таблиц, которые часто приводят к ошибкам в планировании и финансовым потерям.

Продукт решает ключевую проблему администраторов квест-комнат: отсутствие единого, наглядного и надёжного инструмента для учёта занятости комнат, работы мастеров, финансов и истории взаимодействия с клиентами.

Отличительная особенность «Квест-Мастера» — полная автономность (работа без интернета), визуальное недельное расписание с защитой от двойного бронирования и встроенная система финансовой отчётности. В отличие от общих календарей или табличных редакторов, это приложение заточено под специфичные нужды индустрии развлечений.

Установка

Скачайте готовое приложение по ссылке: https://disk.yandex.ru/d/w3EToDX1ll1ARg
Установите APK-файл на устройство с Android 7.0 или новее.

Для разработчиков:

Убедитесь, что у вас установлена Android Studio.

Клонируйте репозиторий на ваш компьютер.

Откройте папку QuestMaster как проект в Android Studio.

Дождитесь завершения индексации и синхронизации Gradle-файлов.

Подключите Android-устройство с включённой опцией "Отладка по USB" или создайте виртуальное устройство (AVD) через меню Tools → Device Manager.

Использование

После установки APK-файла приложение готово к работе.

Краткий старт:

При запуске вы увидите стартовый экран, а затем — Главное меню с пятью разделами.

Для начала работы необходимо добавить квест-комнаты и мастеров через соответствующие разделы.

Чтобы создать бронирование, перейдите в раздел «Бронирование», выберите неделю и время в сетке расписания.

Важно: Система не позволит создать два бронирования на одну комнату в одно время.

Финансовые отчёты формируются автоматически в соответствующем разделе.

Разработка

Проект открыт для изучения и использования в учебных целях. Исходный код написан на Java с использованием архитектурных компонентов Android.

Технологический стек:

Язык: Java

Минимальная версия Android: API 24 (Android 7.0)

База данных: Room Persistence Library (SQLite)

Основные компоненты: Activity, RecyclerView/ListView, CardView

Многопоточность: ExecutorService
